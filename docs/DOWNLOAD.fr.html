Télécharger DaisyDuck

DaisyDuck est disponible pour Windows et pour GNU/Linux.

DaisyDuck pour Windows (la version Linux est plus récente)

Pour télécharger l'installateur : daisyduck-win32-installer-1.0.1.exe

Cette version a été compilée à l'aide de MinGW et embarque les bibliothèques suivantes :

Bibliothèque Version Licence Site
FFmpeg r23791 GPL v2+ http://www.ffmpeg.org
libduck 1.0.0 LGPL v2.1+ page "Développement"
libiconv 1.13.1 LGPL v2+ http://www.gnu.org/software/libiconv/
Qt 4.6.2 LGPL v2.1 http://qt.nokia.com
libxml2 2.7.7 MIT http://xmlsoft.org
libVLC 1.1.0 GPL v2+ http://www.videolan.org

Pour télécharger l'archive source complète (7-Zip) : daisyduck-full-dist-1.0.1.7z
