Download DaisyDuck

DaisyDuck is available for Windows and for GNU/Linux.

DaisyDuck for Windows (the Linux version is newer)

To download the installer : daisyduck-win32-installer-1.0.1.exe

This version was built with the help of MinGW and embarks the following libraries :

Library Version License Site
FFmpeg r23791 GPL v2+ http://www.ffmpeg.org
libduck 1.0.0 LGPL v2.1+ Page "Development"

Qt 4.6.2 LGPL v2.1 http://qt.nokia.com
libxml2 2.7.7 MIT http://xmlsoft.org
libVLC 1.1.0 GPL v2+ http://www.videolan.org

To download the full source package (7-Zip) : daisyduck-full-dist-1.0.1.7z
